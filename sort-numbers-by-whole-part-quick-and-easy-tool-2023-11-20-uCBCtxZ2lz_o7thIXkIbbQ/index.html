How It Works

The Sort by Whole Part tool operates on a simple principle: it focuses on the whole number portion of each value you input. Here’s the breakdown:

  1. You enter a list of numbers (e.g., 3.2, 1.5, 2.9, 4.1).
  2. The tool extracts the whole number part of each (e.g., 3, 1, 2, 4).
  3. It then sorts the numbers based on these whole parts, ignoring the decimal portions.
  4. The final result is a neatly ordered list, grouped by whole numbers.

For example, if you input 3.2, 1.5, 2.9, 4.1, the tool sorts them as 1.5, 2.9, 3.2, 4.1.
