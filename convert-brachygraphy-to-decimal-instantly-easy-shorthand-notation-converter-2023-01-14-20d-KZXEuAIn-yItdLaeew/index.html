How It Works

The tool works by mapping each Roman numeral (brachygraphy) symbol to its corresponding decimal value. Simply input a valid sequence of Roman numerals, and the tool adds up the values to give you the decimal equivalent. For example:

  • "Ⅶ" (7) + "Ⅹ" (10) = 17
  • "Ⅿ" (1000) + "Ⅽ" (100) + "Ⅹ" (10) + "Ⅶ" (7) = 1117
