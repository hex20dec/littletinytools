How It Works

Converting hexadecimal to Mayan numerals might sound complicated, but it’s actually pretty straightforward. Here’s the breakdown:

  1. Start by entering a valid hexadecimal number (using digits 0-9 and letters A-F).
  2. The tool converts the hex number into a decimal (base-10) number.
  3. The decimal number is then translated into the Mayan numeral system, which uses a base-20 (vigesimal) system.
  4. The result is displayed using Mayan symbols: • for 1 and — for 5.

For example, the hexadecimal number 1A3F converts to 6719 in decimal, which then translates into Mayan numerals as a combination of bars and dots. Cool, right?
