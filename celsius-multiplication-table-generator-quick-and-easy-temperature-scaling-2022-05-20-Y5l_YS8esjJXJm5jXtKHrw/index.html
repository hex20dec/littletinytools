How It Works

It’s simpler than you think! The tool takes your starting Celsius value and multiplies it by each number in your chosen range. For example, if you start with 10°C and set a range of 5, it will calculate 10 x 1, 10 x 2, all the way up to 10 x 5. The results are neatly displayed in a table, so you can see the progression at a glance.
