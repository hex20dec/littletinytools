How It Works

This tool is as simple as counting tally marks but way cooler. Here’s how it works:

  1. You enter your tally marks using the pipe symbol | (e.g., |||| ||||).
  2. The tool counts each | as one tally mark.
  3. It then converts the total count into its hexadecimal equivalent.

For example, if you type |||| ||||, the tool counts 8 tally marks and converts it to 8 in hexadecimal. But if you type |||| |||| |||| ||||, it counts 16 tally marks and converts it to 10 in hexadecimal (because 16 in decimal is 10 in hexadecimal). Easy, right?
