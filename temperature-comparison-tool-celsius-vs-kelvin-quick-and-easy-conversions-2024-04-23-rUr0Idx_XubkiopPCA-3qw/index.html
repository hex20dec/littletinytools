How It Works

The tool uses a simple formula to convert temperatures between Celsius and Kelvin. If you input a temperature in Celsius, it adds 273.15 to get the Kelvin equivalent. Conversely, if you input a temperature in Kelvin, it subtracts 273.15 to get the Celsius equivalent. Here's the formula in plain terms:

Kelvin = Celsius + 273.15
