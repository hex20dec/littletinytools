How It Works

The Mayan Numeral to Hexadecimal Converter works like this:

  1. Each dot (•) in the Mayan numeral represents 1.
  2. Each bar (—) represents 5.
  3. A shell (⊚) stands for 0.
  4. The tool adds up all the values to convert the Mayan numeral into a decimal (base-10) number.
  5. Finally, it converts the decimal number into hexadecimal (base-16), which is commonly used in computing and programming.
