How It Works

The tool calculates the temperature range by subtracting the minimum temperature from the maximum temperature. It’s as simple as:

Range = Maximum Temperature - Minimum Temperature

For example, if the minimum temperature is 10°C and the maximum is 25°C, the range is 15°C. The tool also provides a visual representation of the range using a progress bar, making it easy to understand at a glance.
