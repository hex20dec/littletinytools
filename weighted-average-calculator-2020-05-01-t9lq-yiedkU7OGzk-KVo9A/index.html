How the Calculator Works

The Weighted Average Calculator uses a simple formula to calculate the weighted average of a set of numbers. The formula is: Weighted Average = (Value1 * Weight1 + Value2 * Weight2 +... + ValueN * WeightN) / (Weight1 + Weight2 +... + WeightN).

For example, if we have three values (10, 20, 30) with corresponding weights (2, 3, 1), the weighted average would be: (10 * 2 + 20 * 3 + 30 * 1) / (2 + 3 + 1) = (20 + 60 + 30) / 6 = 110 / 6 = 18.33.
