Mayan numerals use symbols for 0 (⊚), 1 (•), and 5 (—). Example: 19 is represented as four dots over three bars.

The Mayan numeral system is a base-20 (vigesimal) system. It uses combinations of dots (1), bars (5), and a shell symbol (0) to represent numbers.

This tool works by converting Mayan numeral symbols into their decimal equivalents using a simple formula. Here’s how it breaks down:

  • • (dot) = 1
  • — (bar) = 5
  • ⊚ (shell) = 0

The tool adds up the values of each symbol in your input to calculate the total decimal number. For example, if you enter •••—, it translates to 3 dots (3 x 1) + 1 bar (5), which equals 8.
