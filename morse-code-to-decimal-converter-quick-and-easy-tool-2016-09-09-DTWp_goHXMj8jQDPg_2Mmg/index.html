Enter Morse code (dots and dashes) to convert it into a decimal number.

Only dots (.), dashes (-), and spaces are allowed.
Supports Morse code for numbers 0-9 only.
How Does It Work?

The tool works by matching the Morse code you input to its corresponding decimal number. Each Morse code sequence for numbers 0-9 is predefined, and the converter simply looks up your input in its database. For example, the Morse code ...-- (dot-dot-dot-dash-dash) translates to the number 3. It’s like having a cheat sheet, but without the flipping through pages!
