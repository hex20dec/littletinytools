How it Works

The Round Kelvin Value Tool uses a simple formula to round the input Kelvin value to the desired number of decimal places. The formula is as follows: rounded value = (input value * 10^decimal places) / 10^decimal places, where the rounding mode (round to nearest, round up, or round down) is applied to the result.
