How It Works

The tool uses a simple formula to map decimals to musical notes. The chromatic scale consists of 12 notes, from C to B. Each note corresponds to an equal slice of the decimal range between 0 and 1. For example:

  • 0 to 0.083 → C
  • 0.083 to 0.166 → C#
  • 0.166 to 0.25 → D
  • ...and so on, up to B at 1.

The tool calculates which slice your decimal falls into and reveals the corresponding note. It’s like turning numbers into a mini symphony!
