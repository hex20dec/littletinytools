How it Works

The Temperature Subtraction Memory Tool uses a simple formula to calculate the result of subtracting one temperature from another in Kelvin. The formula is: Result = Initial Temperature - Temperature to Subtract. For example, if the initial temperature is 100 K and the temperature to subtract is 20 K, the result would be 80 K.
