How it Works

The CSV Custom Delimiter Converter uses a simple yet effective formula to convert your CSV files. The process involves the following steps:

  • Detecting the current delimiter used in the CSV file
  • Replacing the current delimiter with the new delimiter selected by the user
  • Generating the converted CSV file with the new delimiter

The conversion process can be represented by the following formula:

Converted CSV = Original CSV x (New Delimiter / Current Delimiter)

For example, if you have a CSV file with a comma (,) delimiter and you want to convert it to a semicolon (;) delimiter, the formula would be:

Converted CSV = Original CSV x (; /,)
